# N M Amlan — Portfolio

Personal portfolio site (Vite + React + Tailwind CSS + Framer Motion).

## Setup

```bash
npm install
npm run dev
```

## Edit content

All copy and links live in **`src/data/portfolio.js`**.

- Set **`resumeUrl`** to a PDF or view link to show the **Resume** button in the hero.
- Set **`internshipCertificateUrl`** to your Drive folder for the experience section link.
- Replace **`email`**, **`phone`**, **`github`**, **`linkedin`**, and swap **`profileImage`** with your photo (e.g. put `photo.jpg` in `public/` and use `/photo.jpg`).

## Build

```bash
npm run build
npm run preview
```
